# Build Dependencies for RPM Simulator AppImage
# These are the system packages required to build the AppImage

# Build Tools
# - gcc, g++, make, cmake
# - git for source code management
# - pkg-config for library detection

# SDL2 Development Libraries
# Ubuntu/Debian: libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev
# Fedora/RHEL: SDL2-devel SDL2_image-devel SDL2_mixer-devel SDL2_ttf-devel
# Arch: sdl2 sdl2_image sdl2_mixer sdl2_ttf
# openSUSE: libSDL2-devel libSDL2_image-devel libSDL2_mixer-devel libSDL2_ttf-devel

# OpenGL Development Libraries
# Ubuntu/Debian: libgl1-mesa-dev libglu1-mesa-dev
# Fedora/RHEL: mesa-libGL-devel mesa-libGLU-devel
# Arch: mesa
# openSUSE: Mesa-libGL-devel Mesa-libGLU-devel

# X11 Development Libraries
# Ubuntu/Debian: libx11-dev libxext-dev libxrandr-dev libxinerama-dev libxcursor-dev libxi-dev
# Fedora/RHEL: libX11-devel libXext-devel libXrandr-devel libXinerama-devel libXcursor-devel libXi-devel
# Arch: libx11 libxext libxrandr libxinerama libxcursor libxi
# openSUSE: libX11-devel libXext-devel libXrandr-devel libXinerama-devel libXcursor-devel libXi-devel

# Python (for build scripts)
# - python3, python3-pip

# Additional Tools
# - wget (for downloading AppImageTool)
# - file (for file type detection)
# - desktop-file-utils (for desktop file validation)
# - imagemagick (optional, for icon conversion)

# Runtime Dependencies (bundled in AppImage)
# - SDL2 runtime libraries
# - OpenGL runtime libraries
# - X11 runtime libraries

# Note: Run ./scripts/setup_dependencies.sh to install all required packages
# for your specific Linux distribution.
