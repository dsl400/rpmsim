



Task: Set up the virtual environment for running the simulation

Description:
Create a virtual environment with all necessary dependencies for running the simulation.

Steps to complete:
1. Create a new virtual environment in the rpmsim directory
2. Install MicroPython (this might need special handling as it's not available on PyPI)
3. For LVGL, follow platform-specific instructions:
   - On Linux: sudo apt-get install liblvgl-dev
   - On Windows/Mac: Follow instructions at https://lvgl.io/master/get-started/platforms
4. Install pygame package which includes SDL bindings (pip install pygame)

User acceptance criteria:
- A virtual environment should exist in the rpmsim directory
- The virtual environment should have all necessary dependencies installed
