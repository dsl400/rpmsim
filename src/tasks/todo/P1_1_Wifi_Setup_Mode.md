
# Task: Implement Wifi Setup Mode

## Description
Implement the wifi setup mode that is displayed after power on if the device has not been configured yet.

## Requirements
- Create a screen for wifi setup mode
- Display available wifi networks
- Allow user to select a network and enter password
- Show spinner while connecting to wifi
- Check for firmware updates after successful connection

## User Acceptance Test
1. Power on the device without prior configuration
2. Verify that the wifi setup screen is displayed
3. Verify that available wifi networks are shown
4. Select a network, enter password and connect
5. Verify spinner is displayed during connection
6. Verify firmware update check after successful connection
